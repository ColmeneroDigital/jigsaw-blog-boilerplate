# Jigsaw Blog Boilerplate
A preconfigured boilerplate for a [Jigsaw](http://jigsaw.tighten.co/)-generated blog.

## Features / Roadmap

- [ ] Ready-to-use blog
- [ ] Favicons and device icons
- [x] Bootstrap framework
- [x] SEO: Sitemap.xml
- [ ] SEO: Meta tags
- [x] SEO: Robots.txt
- [ ] PWA: Web Manifest
- [ ] PWA: Service Worker
